# iconvwadw-rs
Migrating iconvwadw project C++ to Rust

### icon-browser-adw allows you to view the gnome desktop icons with adwaita, allowing you to copy the icon names for use in your projects.

[![icon-browser-adw](https://snapcraft.io/icon-browser-adw/badge.svg)](https://snapcraft.io/icon-browser-adw)

# Get Ubuntu 24.04 LTS
[![Get it Launchpad](https://launchpadlibrarian.net/606381979/CoF%2064px.png)](https://launchpad.net/~rsvzz/+archive/ubuntu/iconvwadw)

    sudo add-apt-repository ppa:rsvzz/iconvwadw
    sudo apt update
    sudo apt install iconvwadw

# Requirements
    libgtk-4-1 >= 4.14
    libadwaita-1 >= 1.5
    librsvg2-2 >= 2.58
    adwaita-icon-theme
